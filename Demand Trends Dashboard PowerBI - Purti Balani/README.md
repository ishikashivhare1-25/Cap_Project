# Urban Mobility Demand Trends Dashboard

## Project Overview
Interactive Power BI dashboard analyzing ride demand trends, operational performance, customer behavior, cancellations, and revenue intelligence.

## Features
- Dynamic KPI trend analysis
- Multivariate analysis
- Interactive filtering
- Operational intelligence monitoring
- Revenue analytics
- Customer behavior analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
- GitHub

## Dataset
The dataset contains:
- Booking status
- Pickup & drop locations
- Vehicle type
- Booking value
- Ride distance
- Ratings
- Cancellation details
- VTAT & CTAT
- Payment methods
- Time attributes

## Key KPIs
- Total Bookings
- Completed Rides
- Completion Rate
- Ride Incomplete
- Total Revenue
- Avg Booking Value
- Avg Ride Distance
- Avg VTAT
- Avg CTAT
- Ride Efficiency Score

## Dashboard Features
### Dynamic KPI Trend Analysis
Users can dynamically select KPIs and analyze trends over time.

### Multivariate Analysis
Dynamic X-axis and Y-axis parameter selection enables interactive relationship exploration.

### Operational Intelligence
Analyze:
- Ride cancellations
- Incomplete rides
- Operational delays
- Ride fulfillment efficiency
